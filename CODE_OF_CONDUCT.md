# Código de Conducta para ND-Toolkit

## Nuestro compromiso
Nos comprometemos a hacer de este proyecto un espacio seguro, inclusivo y respetuoso para todas las personas, independientemente de su neurotipo, identidad, edad o procedencia.

## Comportamiento esperado
- Usar lenguaje inclusivo y respetuoso
- Validar las experiencias de otras personas ND
- Aceptar críticas constructivas con apertura
- Centrarse en lo que es mejor para la comunidad

## Comportamiento inaceptable
- Capacitismo, insultos o comentarios despectivos
- Invalidar la experiencia de otra persona ND
- Acoso, doxxing o compartir información privada
- SPAM o promoción de productos dañinos

## Consecuencias
El incumplimiento de este código puede conllevar la expulsión del proyecto y sus canales.
